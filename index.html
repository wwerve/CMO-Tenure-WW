TL;DR

Good — with one unresolved question

The CMO came in, pipeline went up 63%, the team now measures everything — events, paid media, brand — and program spend pays itself back within a quarter. That's a strong two years.

The unresolved question is whether all that pipeline is going to convert. Bookings only went up 28% while pipeline went up 63%, and the 2026 budget just jumped 62% on top of that. If the conversion problem — UK SQOs, ENT volume, Install Base win rates — gets fixed, this is an easy story. If it doesn't, you're spending more into a funnel with a leak.

Green on marketing. Amber on the business overall. The CMO built the machine. Whether it pays out depends on something she doesn't fully control.

02 — Pipeline vs Bookings Gap

Not a marketing problem

Pipeline grew 63%. Bookings grew 28%. In Q1 2026, $4.8M left the pipeline as closed lost or unidentified — 31% of the opening balance.

"The gap is concentrated in UK Marketing, ENT volume, and Install Base conversion — all post-marketing handoff."
UK Marketing SQO failure — critical
1 SQO generated vs 52 budgeted in Q1 2026 (−98%). UK Sales compensating but unsustainably. Urgent diagnostic required before adding headcount.
Install Base win rate
Pipeline at 119% of target — leads are healthy. Win rate actuals appear negative in the data (verify against raw deals — may be data quality issue). Conversion layer needs investigation.
ENT marketing soft volume
Pipeline at 34% of target ($583K vs $1.65M). Only 7 SQOs vs 36 budgeted. Win rates hold ~45–50% when deals close — the problem is SQO volume, not conversion quality.

07 — Recommendations

Six actions, three priorities

01
Diagnose UK Marketing SQO failure Urgent

1 SQO vs 52 budgeted in Q1 2026. Run diagnostic before adding any UK headcount. UK Sales compensating but unsustainably.

02
Audit Install Base win rates Urgent

Win rate actuals appear negative — mathematically suspect. Verify against raw deal data before treating as a confirmed crisis. Resolve the data quality question first.

03
Put Empower ($200K) on watch High

ComplyConnect 2025 swing deal, close date March 2027. Assign a named owner now.

04
Set ComplyConnect 2026 cost ceiling at $220K High

At $265K the 2025 event was marginally ROI-positive. Tighter production, same experience.

05
Require SF attribution for all events Medium

Advyzon and ComplyConnect both prove measurable ROI under tracking. Apply the same standard to enterprise dinners and box seats/sports events.

06
Run 6Sense ENT pipeline attribution Medium

$147K ABM spend vs 7 ENT SQOs in Q1 2026. Validate ROI before renewing.

Overall verdict: The CMO has built something with genuine ambition and measurable momentum. Pipeline is growing, efficiency is improving, event attribution is rigorous, paid media ROI is now measured, and the Comply rebrand is landing positively on LinkedIn. The conversion gap — pipeline vs bookings, CW ROAS, win rates — requires joint ownership. Two years in, this tenure has earned continued investment and high expectations in equal measure.
